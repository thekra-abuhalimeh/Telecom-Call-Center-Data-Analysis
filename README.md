# Telecom-Call-Center-Data-Analysis
Telecom call center data analysis using Python and Power BI to evaluate performance, efficiency, and customer service metrics.

## Overview
This project analyzes telecom call center data to evaluate operational performance and customer service efficiency. The analysis focuses on identifying patterns in call volume, resolution rates, response times, and customer satisfaction.

## Tools
- Python
- Pandas
- NumPy
- Microsoft Power BI

## Key Metrics
- Total Calls
- Average Satisfaction Rating
- Call Answer Rate
- Call Resolution Rate
- Average Speed of Answer
- Agent Efficiency

## Dashboard Features
- Call center performance monitoring
- Resolution rate by support topic
- Call distribution analysis
- Call trends by hour and month
- Interactive filters for deeper insights
